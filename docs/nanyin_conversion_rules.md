# 南音工ㄨ谱转换规则

这份规则是 App 的核心，不依赖横排或竖排版式。版式只决定读取顺序，真正的转换由“音高表 + 节奏表 + 校验表”完成。

## 1. 版式读取

| 版式 | 读取顺序 | 输入层 | 校验方式 |
| --- | --- | --- | --- |
| 竖排传统谱 | 从右到左分栏，栏内从上到下 | 蓝色工ㄨ谱字、红色拍点、黑色歌词 | 用人工模板检查完整率 |
| 横向对照谱 | 从左到右，按小节和换行继续 | 下方蓝色工尺/工ㄨ谱，上方黑色简谱 | 用页面自带黑色简谱做标准答案 |

说明：横排、竖排只影响排序，不改变“工、六、思、一”等谱字到简谱的对应关系。

## 2. 音高转换

| 工ㄨ谱字 | 简谱 | 五音 | 唱名 | 说明 |
| --- | --- | --- | --- | --- |
| 乂 / ㄨ | 1 | 宫 | Do | 义、又、×、X、尺先作为 OCR/字形兜底 |
| 工 | 2 | 商 | Re | 识别特征是上下横和中间竖 |
| 六 | 3 | 角 | Mi | 与工、士等相似字通过笔画结构区分 |
| 思 / 士 | 5 | 徵 | Sol | 先归为同一音高，异体写法后续继续校准 |
| 一 | 6 | 羽 | La | 要避免和延续线混淆 |

## 3. 节奏转换

| 符号 | 类型 | 播放策略 |
| --- | --- | --- |
| ○ / 〇 / o / O / 。 | 拍 | 强位重音，默认 1 个拍/撩单位 |
| 、 / 丶 | 撩 | 弱位，默认 1 个拍/撩单位 |
| 厶 / ∠ | 撩拍变体 | 按撩拍单位处理，保留原始提示 |
| - / — / \| / │ / ｜ | 延续 | 延长前一个音 |
| √ / ） / ) / L / ㄥ / 口 / 十 | 南琶指骨 | 保留在播放稿；无撩拍标记时先按短促动作估算 |

## 4. 处理流程

1. 判断版式：竖排传统谱或横向对照谱。
2. 按颜色分层：黑色歌词/简谱、蓝色工ㄨ谱、红色拍点。
3. 按版式排序谱字和符号。
4. 用音高表把谱字转换成简谱数字。
5. 用节奏表把拍、撩、延续和指骨转成播放时值与重音。
6. 竖排谱用模板校验完整率；横向对照谱用黑色简谱校验翻译结果。

## 5. 比赛讲法

可以这样讲：

> 我们不是为横排和竖排写两套规则，而是先判断怎么读，再用同一张转换表。竖排谱从右往左读，横向对照谱从左往右读；读出来的工ㄨ谱字都进入同一套“谱字转简谱、符号转节奏”的规则库。
