# 南音识谱比赛展示材料

## 1. 项目一句话

用可解释算法把南音工乂谱转换成可读的简谱和可听的旋律，让同龄人更容易理解南音。

## 2. 作品定位

本项目不是做“南音百科”，也不是比拼通用 OCR，而是聚焦一个具体痛点：

- 年轻人看不懂南音工乂谱。
- 传统谱字和现代简谱之间缺少直观转换。
- 谱子如果只能看不能听，学习门槛仍然较高。

项目目标是完成一个闭环：

```text
谱图输入 → 工乂谱识别 → 简谱翻译 → 符号节奏解释 → 旋律播放
```

## 4. iPad 展示路径

1. 打开 App，默认进入《静夜思》演示谱算法页。
2. 介绍项目算法名称：南音工乂谱字符特征打分匹配法。
3. 切到“流程”，用原图、黑白化、分栏、单字打分四格图讲算法。
4. 切到“符号”，说明拍、撩、延续、指骨如何影响节奏。
5. 切到“简谱”，点击播放，让评委听到旋律。
6. 点击“识别”，展示 OCR 不完整时如何使用校对模板兜底。
7. 切到“材料”，回答项目定位、平台选择和测试计划。

## 5. 2 分钟讲解稿

大家好，我的项目是《南音工乂谱识别与翻译》。

南音是泉州的重要非遗音乐，但它使用的工乂谱和我们平时见到的简谱不一样，很多同学看不懂。所以我想做一个工具，把传统工乂谱转换成现代简谱，并且播放出来。

我的方法是设计一个可解释识谱算法。它先把谱图处理得更清楚，再根据南音谱竖排的特点进行分栏，把单个谱字切出来。然后它会观察每个字的特征，比如横线、竖线、圆形、宽高比例，再给不同候选字打分。

例如“工”字有上下两横和中间竖线，所以它会得到更高分，最后识别为“工”，再转换成简谱 2。南音里的拍、撩、延续等符号不改变音高，但会影响播放时的强弱和时长。

这个 App 可以在 iPad 上展示完整流程：看原谱、看算法、看简谱、听旋律。它让传统南音谱变得更容易被同龄人理解，也用简单算法帮助非遗传播。

## 6. 评委问答准备

### 为什么只展示 iPad/iPhone？

比赛重点是算法流程和文化应用验证，不是商业化多端上线。iPad 展示稳定，屏幕也更适合讲谱图、算法和播放结果。项目的核心规则库不依赖 iOS，后续可以迁移到网页或 Android。

### OCR 不准怎么办？

OCR 不是项目核心。它只是图像输入的一环。项目真正的亮点是南音谱字规则、符号节奏规则和可解释识别流程。演示样例会使用校对模板兜底，保证完整率和播放稳定性。

### 算法体现在哪里？

算法体现在六步：

1. 图像预处理。
2. 竖排谱面分栏。
3. 字符切分。
4. 字符特征提取。
5. 打分匹配。
6. 简谱和节奏转换。

### 为什么适合初一学生？

因为算法不是黑盒模型，而是能讲清楚的规则打分。比如“工”有横线和竖线，“○”有圆形和中间空白，这些特征都能直观看到。

## 10. 横竖谱面统一规则

横排和竖排都支持，但规则不写两套。App 先判断谱面怎么读：竖排从右到左、栏内从上到下；横向对照谱从左到右、按小节和换行继续。读出来的谱字再进入同一套转换规则：工ㄨ谱字转简谱，拍/撩/延续/指骨转播放节奏。

详细规则见 `docs/nanyin_conversion_rules.md`。
