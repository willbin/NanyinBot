这个目录保存和泉州南音网样例相关的本地资料，方便 Git 统一管理。

## 已保存资料

| 文件 | 来源 | 用途 |

## 保存原则

- 文档、截图、人工标注表放进工程并提交 Git。
- 完整外站视频不放进 Git，只在 `docs/qznanyin_reference_samples.md` 记录来源链接和复现方法。
- 新增截图统一命名为 `<曲目拼音或页面名>-reference.png`。
